# 🩺 Medical Trends Tracker

> Персональный дашборд для визуализации динамики медицинских показателей за 5+ лет.  
> *Сделано для себя, упаковано для портфолио.*

## 💡 Зачем это нужно

**Как было** \
При регулярной сдаче анализов медицинские лаборатории выдают результаты в разрозненных PDF-файлах. Отслеживать динамику показателей (например, тренд изменения Ферритина или ТТГ относительно друг друга) на отрезке в ~5 лет вручную через файлы — сложно. Стандартные референсы лабораторий не показывают индивидуальный волнообразный тренд.

**Как стало** \
Все исторические данные оцифрованы и сведены в единую структуру. С помощью интерактивного дашборда можно в один клик оценить корреляцию между разными группами показателей (например, железосвязывающая способность + гормоны) на настраиваемом таймлайне.

**Проблема** \
В ходе проектирования MVP были выявлены технические ограничения визуализации. 

В одном отчете могут быть показатели с разными диапазонами и из разных групп анализов (например, `ЩФ: 40–130 мкмоль/л` и `АЛТ: 0–32 нг/мл`). Если вывести всё на один график — мелкие значения «теряются».

## 🎯 Как я это решила
Для решения этих задач я разработала архитектуру дашборда, для которой заложила следующие требования:

| Проблема | Решение | Роль аналитика |
|----------|---------|---------------|
| Большое количество показателей | Показатели разбиты на 15 логических групп (печеночные пробы, липидный профиль, андрогены и т.д.), чтобы не перегружать один экран. | Декомпозиция домена, классификация |
| Разные масштабы показателей. Внутри одной группы показатели могут критически различаться по значениям (например, Параметр А со значением `1–5` и Параметр Б со значением `100–200`). При стандартном выводе график параметра А превращается в плоскую нечитаемую линию. | Интерактивное скрытие/показ линий на графике. Интегрирована функция *динамического скрытия/отображения* графиков через легенду Plotly. Пользователь может в один клик изолировать нужный показатель. | Проектирование UX, приоритизация |
| Управление временными диапазонами (Dynamic Filters): Сдавать много анализов в течение 5 лет означает получить перегруженную горизонтальную ось. | Добавлен сквозной JS-фильтр по годам («С:» / «ПО:») для отсечения неактуальных исторических периодов. Фильтр по диапазону лет с синхронизацией всех графиков. | Формулировка требований к фильтрации |
| Данные в 100+ PDF-файлах | Парсинг с помощью ИИ → структурированный DataFrame | Моделирование данных, нормализация |

## 🔍 Аналитическая рефлексия

На старте думала: «просто построю графики».  
Но в процессе поняла:
- Без группировки по типам показателей — визуальный шум
- Без возможности скрыть метрики — невозможно сфокусироваться на важном
- Без фильтра по времени — сложно отслеживать долгосрочные тренды

Это типичный аналитический путь: от сырой потребности → к структурированным требованиям → к проверяемому решению.
