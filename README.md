# 🩺 Medical Trends Tracker

> Персональный дашборд для визуализации динамики медицинских показателей за 5+ лет.  
> *Сделано для себя, упаковано для портфолио.*

## 💡 Зачем это нужно

**Как было:** При регулярной сдаче анализов медицинские лаборатории выдают результаты в разрозненных PDF-файлах. Отслеживать динамику показателей (например, тренд изменения Ферритина или ТТГ относительно друг друга) на отрезке в 5 лет вручную через файлы — невозможно. Стандартные референсы лабораторий не показывают индивидуальный волнообразный тренд.
* **Как стало:** Все исторические данные оцифрованы и сведены в единую структуру. С помощью интерактивного дашборда можно в один клик оценить корреляцию между разными группами показателей (например, железосвязывающая способность + гормоны) настраиваемом таймлайне.

**Проблема**: в одном отчете могут быть показатели с разными диапазонами и из разных групп анализов (например, `ЩФ: 40–130 мкмоль/л` и `АЛТ: 0–32 нг/мл`). Если вывести всё на один график — мелкие значения «теряются».

## 🎯 Как я это решила

| Проблема | Решение | Роль аналитика |
|----------|---------|---------------|
| Разные масштабы показателей | Группировка по клиническим категориям (печень, почки, железо, витамины и т.д.) | Декомпозиция домена, классификация |
| «Шум» от нерелевантных метрик | Интерактивное скрытие/показ линий на графике | Проектирование UX, приоритизация |
| Нужно сравнивать периоды | Фильтр по диапазону лет с синхронизацией всех графиков | Формулировка требований к фильтрации |
| Данные в 114 PDF-файлах | Ручной парсинг → структурированный DataFrame | Моделирование данных, нормализация |
